PRODUCT DOCUMENTATION: CloudStore Pro v3.5
Acme Cloud Solutions | Technical Reference Guide

OVERVIEW
CloudStore Pro is an enterprise-grade cloud storage and collaboration platform. It supports real-time document editing, version control, and role-based access management for teams of all sizes.

SYSTEM REQUIREMENTS
- Operating System: Windows 10/11, macOS 12+, Ubuntu 20.04+
- RAM: Minimum 4 GB (8 GB recommended)
- Disk Space: 2 GB for desktop client
- Internet: 10 Mbps minimum, 50 Mbps recommended
- Browsers: Chrome 110+, Firefox 110+, Edge 110+, Safari 16+

INSTALLATION
Desktop Client:
1. Visit https://cloudstore.acmecorp.com/download
2. Select your operating system and download the installer.
3. Run the installer and follow the setup wizard.
4. Sign in with your company email (SSO supported).

Mobile App:
- iOS: Available on the App Store (search "CloudStore Pro")
- Android: Available on Google Play Store

FEATURES

File Storage & Sync
- Unlimited cloud storage (subject to your plan).
- Real-time sync across all devices.
- Offline access: mark files for offline availability.
- Automatic versioning: retains last 100 versions of each file.

Collaboration
- Real-time co-editing for documents, spreadsheets, and presentations.
- Comments and annotations on any file type.
- @mentions in comments to notify team members.
- Shared drives for team-level access.

Security
- End-to-end encryption (AES-256) for all files.
- Two-factor authentication (2FA) required for all enterprise accounts.
- Admin-controlled access policies and audit logs.
- GDPR and SOC 2 Type II compliant.

ACCOUNT MANAGEMENT

Password Reset:
1. Go to https://cloudstore.acmecorp.com/login
2. Click "Forgot Password?"
3. Enter your registered email address.
4. Check your email for the reset link (valid for 30 minutes).
5. Create a new password (minimum 12 characters, must include uppercase, number, and symbol).

Managing Storage:
- Dashboard → Storage → View Usage to see your current usage.
- Upgrade your plan from Dashboard → Billing → Change Plan.
- Delete old versions from File → Version History → Delete Selected.

TROUBLESHOOTING

Sync Issues:
- Ensure the desktop client is up to date.
- Check internet connectivity.
- Quit and restart the CloudStore app.
- If sync stalls, right-click the file → Force Sync.

Login Problems:
- Clear browser cache and cookies.
- Disable browser extensions temporarily.
- Contact IT support if SSO fails: it-support@acmecorp.com
